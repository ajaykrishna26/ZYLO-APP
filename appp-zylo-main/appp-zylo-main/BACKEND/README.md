# Dyslexia Reading Assistant API
This is the backend API for the Dyslexia Reading Assistant, hosted on Hugging Face Spaces using Docker.

## Setup
Ensure you have added the `MONGO_URI` and `JWT_SECRET_KEY` secrets in the Space settings.
